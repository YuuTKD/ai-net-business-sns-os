## P1: AI店舗集客診断レポート（B2B_AGENT）

### 概要

SNSで告知 → Google Forms無料診断 → Claude Codeがレポート草案生成（5分）→
人間レビュー・送付（15分）→ 有料詳細レポート案内（¥1,980）→ 30分診断提案（¥3,000〜）

### 診断フォーム設計（同意＋5問）

```
【同意欄】フォーム冒頭に必ず設ける（Google Forms「説明」欄を使用）
  ✅ 入力いただいた情報は診断レポート作成のみに使用します
  ✅ 第三者への提供・販売は行いません
  ✅ 氏名・店舗名は診断結果送付に使用し、他の用途には使いません
  → チェックボックス必須項目として設定（未チェックは送信不可）

Q1. 現在使っているSNS / 投稿頻度は？
    （Instagramを週○回 / X（旧Twitter）を週○回 / やっていない）

Q2. Google Business Profile（Googleマップ）を更新していますか？
    （毎週更新 / たまに / ほとんどしていない）

Q3. 口コミへの返信はしていますか？
    （全件返信 / 時々 / ほとんどしていない）

Q4. 集客の一番の悩みは何ですか？（自由記述）

Q5. 店舗の業種・規模は？
    （美容 / 飲食 / 整体 / その他、スタッフ人数）
```

### 自動化スコープ（Claude Code担当）

```
[ ] 診断フォーム設計書（Google Forms設計用Markdown）生成
[ ] レポートテンプレート設計（フォーム回答 → レポート差し込み用）
[ ] 個別レポート草案生成（フォーム回答CSV → Markdown）
[ ] SNS告知文・DM文案生成（手動送信前に確認）
[ ] 有料レポートのコンテンツ草案生成
```

### 人間確認（ゆうさん担当）

```
[ ] Google Forms作成・公開
[ ] レポート草案の内容確認・送付（15分/件）
[ ] 有料レポートのBrain公開
[ ] 30分診断の日程調整（Google Calendar）
[ ] 診断実施（30分）
```

### コンプライアンス

- 「診断結果は改善提案であり、集客・収益を保証するものではありません」を必ず記載
- 「医療・法的アドバイスではありません」を明記（整体院等向けの場合）
- 個人情報（店名・オーナー名）はサービス提供のみに使用
