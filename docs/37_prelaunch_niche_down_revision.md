## 変更理由

### なぜ "Client Work" から "Client Docs" に変えるか

**理由1：Docsの方が具体的**
「Work」は何でも指しうる抽象語。
「Docs」はプロポーザル、ヒアリングシート、メッセージフレームワークなど、
実際の手元にある資料を即座に連想させる。

購入者の頭の中で「これ、自分のGoogleドライブにある」と
一瞬で想像できるかどうかが、コンバージョン率を左右する。

**理由2：コピーライター・マーケター層との相性**
コピーライターとソロマーケターはドキュメントベースで仕事をしている。
提案書、ヒアリングシート、ブランドボイスガイド——
これらはすべて「docs」として存在する。
"Work" という言葉より "Docs" の方が彼らの作業実態に近い。

### なぜ "AI" をタイトルから外すか

**理由1：AIをタイトルに入れると安っぽく見える**
2024〜2025年以降、"AI" をタイトルに含む商品は市場に溢れている。
"Turn X into Y with AI" 系の商品名はすでに差別化にならない。

**理由2：主役はユーザーの資料、AIは手段**
ユーザーが価値を感じるのは「自分の提案書が売り物になる」という事実。
AIは手段であり、タイトルで強調するものではない。
サブタイトルと商品内容でAI活用を伝えれば十分。

**理由3：No hype トーンの徹底**
"AI" をタイトルに入れると「またAI副業系か」という
第一印象のフィルタをかいくぐれない。
"Sell the Client Docs You Already Use" は、
誇張ゼロ・具体的・すぐ使えるというメッセージを静かに伝える。

### なぜターゲットをcopywriters / solo marketersに絞るか

**理由1：初回ローンチでは広げすぎない**
全職種のフリーランサーを対象にすると、
投稿トーン・商品内容・診断クイズのすべてが「誰にでも当てはまる曖昧さ」になる。

最初の5〜10件の販売は、
自分のポジションを最も鮮明に伝えられるターゲットに集中して取るべき。

**理由2：コピーライター・マーケター層はLinkedIn / Threadsに多い**
この層はLinkedInで積極的に発信・交流しており、
「同業者の知恵からお金を払って学ぶ」ことへの抵抗が低い。
$19であれば衝動買いしやすい層でもある。

**理由3：商品内容がコピーライター層にそのまま刺さる**
提案書・ヒアリングシート・メッセージフレームワーク——
これらはコピーライターとソロマーケターが実際に使っている資料そのもの。
「これ、自分のことだ」という共鳴が生まれやすい。

**理由4：絞っても市場は十分に広い**
英語圏のフリーランスコピーライターとソロマーケターは
グローバルで数十万人規模。
最初の5件販売に必要なのは数百人のリーチで十分。
