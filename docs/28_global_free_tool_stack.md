## 無料で始める場合の推奨構成

```
販売     ：Gumroad（手数料10%のみ）
メール   ：Brevo（連絡先無制限・無料）
診断     ：Tally（回答無制限・無料）
LP       ：Notion Sites または Carrd（無料プラン）
集客     ：X + Threads（無料）
制作     ：Google Docs + Google Sheets + Canva（無料）
管理     ：Google Sheets（無料）
AI       ：Claude Code + ChatGPT（既存）

合計初期費用：$0
```
