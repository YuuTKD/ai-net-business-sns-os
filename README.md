# AI Net Business SNS OS

このプロジェクトは、InstagramとThreadsを使ってAIネットビジネスを収益化するためのSNS自動化OSです。

## 目的

- SNS投稿の自動生成
- Threads自動投稿
- Instagram運用の半自動化
- 投稿インサイト分析
- 勝ち投稿の再利用
- LINE/メルマガ/LP/商品導線への接続
- AIテンプレ販売・AI自動化支援・コンサル獲得

## 注意

- 店舗事業の YU Business OS とは分離管理する
- Secret / token / credentials は絶対にコミットしない
- pushは明示指示があるまでしない
- まずは設計とシート構成から開始する
