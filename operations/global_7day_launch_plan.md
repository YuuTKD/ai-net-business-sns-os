# グローバル7日間ローンチプラン

## 目標

7日間でGumroad商品ページを立ち上げ、最初の1件販売を目指す。

---

## Day 1（月曜）：ツール登録

- [ ] Gumroad登録
- [ ] Brevo登録
- [ ] Tally登録
- [ ] Beacons登録
- [ ] Xプロフィール英語化（Bio・リンク設定）
- [ ] Claude Codeに30-Day Launch Kit本文作成を依頼

## Day 2（火曜）：商品作成

- [ ] Google Docsで30-Day Launch Kit本文確認・調整
- [ ] Canvaで商品表紙画像作成（A4 / PNG）
- [ ] Gumroadに商品登録（$19・非公開で設定）
- [ ] Claude Codeに診断クイズ10問を依頼

## Day 3（水曜）：診断・LP作成

- [ ] TallyでAI Digital Product Readiness Quiz作成（Claude Code提供の質問文使用）
- [ ] CarrdまたはNotionでLP作成（Claude Code提供のコピー使用）
- [ ] Claude CodeにBrevo 7日間メール文を依頼

## Day 4（木曜）：メール設定・公開

- [ ] Brevo 7日間ステップ配信を設定
- [ ] TallyとBrevoの連携設定
- [ ] Gumroadを公開状態に変更
- [ ] Beaconsにリンクをまとめて設置

## Day 5（金曜）：初投稿

- [ ] X / Threadsに1本目投稿（気づき型・商品の宣伝ではない）
- [ ] プロフィールにTally診断URLを設置確認
- [ ] 投稿後のエンゲージメントを確認

## Day 6（土曜）：2本目投稿・確認

- [ ] 2本目投稿（スタック紹介型）
- [ ] 診断回答者の有無確認
- [ ] Brevoでメール配信テスト送信

## Day 7（日曜）：振り返り・翌週設計

- [ ] 1週目KPI記録（投稿数/診断回答数/メール登録数）
- [ ] 初販売の有無確認
- [ ] Week 2の投稿テーマ10本をClaude Codeに依頼
