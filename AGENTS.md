# AGENTS.md — ai-net-business-sns-os

Codex（OpenAI）エージェントが読む指示ファイル。
Claude Code と連携して動作する際のルールを定義する。

---

## エージェントの役割（スコープ）

Codex エージェントが担当できるタスク:

- コンテンツ案の生成（SNS投稿文・メール文・商品説明文）
- タスクの分解・TASK.md へのエントリ追加提案
- ドキュメントのドラフト作成
- コードのレビュー・提案（実行は Claude Code が行う）

Codex エージェントが担当**できない**タスク:

- 外部API の直接呼び出し
- ファイルシステムへの書き込み（提案のみ、実行は Claude Code）
- Secret / 環境変数の参照・生成
- デプロイ・Scheduler 操作
- 本番SNS投稿・自動DM送信

---

## 出力形式のルール

| 項目 | ルール |
|------|--------|
| 言語 | 日本語優先（英語コンテンツは英語で出力） |
| フォーマット | Markdown |
| ファイル提案 | 既存ファイルの差分形式で示す（全文貼り付け不可） |
| タスク提案 | TASK.md フォーマットに合わせて出力 |
| 報告提案 | REPORT.md フォーマットに合わせて出力 |

---

## Claude Code との連携フロー

```
Codex: コンテンツ案・タスク案・コード案を生成
    ↓
Claude Code: 内容を確認し、必要なファイル操作・実行を行う
    ↓
ゆうさん: PR でレビュー・承認
    ↓
Claude Code: merge
```

Codex は提案にとどめ、実行判断は Claude Code とゆうさんが行う。

---

## 禁止行為

- `.env.local` 参照の提案・実行
- Secret を含むコード生成
- 自動投稿・自動DM を即時実行するコードの生成
- 既存 Skill / Knowledge の削除提案
- `git add .` を含む操作の提案

---

## このファイルの更新ルール

ゆうさんの承認を得た PR を通じてのみ変更可。
Codex・Claude Code による自律変更は禁止。
