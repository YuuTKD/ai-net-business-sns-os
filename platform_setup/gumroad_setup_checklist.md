# Gumroad Setup Checklist

## Time needed: ~45 minutes

---

## Step 1: Create Product

- [ ] Log in to Gumroad → click **New Product**
- [ ] Select type: **Digital Product**
- [ ] Product name: `30-Day AI Digital Product Launch Kit`
- [ ] Click **Create Product**

---

## Step 2: Add Description

- [ ] Click the **Description** field
- [ ] Copy the full description from:
  `platform_setup/gumroad_launch_kit_listing.md` → Section 3
- [ ] Paste and review formatting
- [ ] Check that bullet points display correctly

---

## Step 3: Set Price

- [ ] Click **Pricing**
- [ ] Set price: `$19`
- [ ] Currency: USD
- [ ] Leave "Pay what you want" OFF

---

## Step 4: Upload Product File

- [ ] First: Complete `products/30_day_ai_digital_product_launch_kit.md`
  and export/save as PDF (Google Docs → File → Download → PDF)
- [ ] Upload the PDF to Gumroad: drag and drop into the **Product files** area
- [ ] Add a note inside the PDF with your KPI Tracker Google Sheets link

---

## Step 5: Upload Cover Image

- [ ] Create cover image in Canva (recommended: 1280×720px)
- [ ] Use the cover text from:
  `platform_setup/gumroad_launch_kit_listing.md` → Section 4
- [ ] Upload to Gumroad cover image area

---

## Step 6: Post-Purchase Message

- [ ] Click **Content** or **Post-purchase**
- [ ] Copy the message from:
  `platform_setup/gumroad_post_purchase_message.md`
- [ ] Replace `{{SUPPORT_EMAIL}}` with your actual email address
- [ ] Paste into the post-purchase message field

---

## Step 7: Add Tags

- [ ] Click **Tags** section
- [ ] Add tags from:
  `platform_setup/gumroad_launch_kit_listing.md` → Section 5
- [ ] Add up to 10 tags

---

## Step 8: Create Discount Code

- [ ] Go to **Discount** tab or create from dashboard
- [ ] Code name: `LAUNCH5`
- [ ] Amount: $5 off
- [ ] Type: Fixed amount
- [ ] Applies to: this product only
- [ ] Status: Active
- [ ] Note: Use in Day 4 Brevo email

---

## Step 9: Publish

- [ ] Set visibility to **Public**
- [ ] Click **Publish** / **Save**
- [ ] Copy the product URL — you will need it for:
  - Brevo emails (replace {{GUMROAD_LINK}})
  - Tally quiz result pages (replace {{GUMROAD_LINK}})
  - Social media posts

---

## Step 10: Test Purchase

- [ ] Create a 100% off code: `TESTBUY`
- [ ] Open product page in incognito/private window
- [ ] Apply code and complete test purchase
- [ ] Verify: you receive the file and the post-purchase message
- [ ] Delete or deactivate TESTBUY code after testing

---

## Final Checklist Before Going Live

- [ ] Product title is correct
- [ ] Price is $19
- [ ] Description looks clean and readable
- [ ] Cover image is uploaded
- [ ] PDF file is uploaded and downloadable
- [ ] Post-purchase message is set
- [ ] LAUNCH5 discount code is created
- [ ] Product URL is copied and saved
- [ ] Test purchase completed successfully

---

## After Publishing

Copy your Gumroad product URL and paste it in these places:

- [ ] Tally quiz result pages (replace all {{GUMROAD_LINK}})
- [ ] Brevo email sequence (replace all {{GUMROAD_LINK}})
- [ ] X / Threads bio
- [ ] Social posts CTA
