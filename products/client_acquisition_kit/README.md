**Week 1:** Set up your outreach and discovery call framework
**Week 2:** Send your first 10 outreach messages using the templates
**Week 3:** Run your first calls using the discovery framework
**Week 4:** Refine based on what worked

You don't need to use all 5 documents on day one.
Start with the Outreach Templates. Get one response. Then use the rest.
