## 口コミ返信を放置すると起きること

口コミへの返信がない状態は、次のお客様に「反応しない店舗」という印象を与えます。

特に以下の状況では、その印象が強くなりがちです。

- 競合他店が丁寧に返信しているとき
- 初めてその店を検討しているお客様が口コミを読んでいるとき
- ネガティブな口コミが返信なしのまま残っているとき

「口コミに返信することで集客が増える」「評価が上がる」とは断言できませんが、  
返信を丁寧に続けることは、**既存客への感謝の表明**として機能します。

## やってはいけないNG返信 3パターン

テンプレートを使う前に、まずこれだけ確認してください。

### NG①：固有名詞を入れ忘れたままコピペする

```
×「[店舗名]へのご来店ありがとうございます」と書いたまま投稿してしまう
```

必ず `[ ]` の箇所を実際の情報に書き換えてから投稿してください。

### NG②：ネガティブ口コミに感情的に返信する

```
×「事実と異なります」「そのようなことはございません」と強く否定する
×「ご来店をお断りします」と返信する
```

感情的な返信は、他のお客様に悪い印象を与えます。  
このキットのカテゴリCでは、誠実な対応例を用意しています。

### NG③：口コミ投稿を誘導・依頼する返信をする

```
×「ぜひまた口コミを書いてください」
×「次回ご来店時にサービスをお渡しします（返信文内で口コミへの返礼を示唆）」
```

口コミの投稿を報酬で誘導する行為はGoogleのポリシーに抵触する可能性があります。  
このキットにはそのような表現は含まれていませんが、カスタマイズ時もご注意ください。

### できること

- 口コミへの返信文をコピー&ペーストで準備できる

- 悪意ある口コミを削除・非表示にすることはできません
- Googleのアルゴリズムに働きかけるものではありません
- 自動返信システムではありません（手動でのご利用が前提です）

## よくある質問

**Q：Google以外の口コミにも使えますか？**  
A：基本的な文体はYelp・食べログ・Googleマップ等に流用できます。プラットフォームごとに文字数制限が異なるためご調整ください。

**Q：テンプレートを使うと、全店舗が同じ文になりませんか？**  
A：`[ ]` の書き換え箇所があり、店舗名・担当者名・具体的な内容を入れるため、仕上がりはそれぞれ違う文章になります。

**Q：スタッフに渡して使えますか？**  
A：ご購入者の店舗内でのご利用は問題ありません。書き換えルール（NG③等）をスタッフと共有してからご使用ください。

**Q：ネガティブ口コミへの返信が怖いです。**  
A：カテゴリCに誠実な返信テンプレートを用意しています。感情的にならず、事実を丁寧に伝えることを基本としています。深刻なトラブルに発展している場合は、専門家（弁護士等）への相談をお勧めします。

**Q：返信した後に効果を確認する方法は？**  
A：Googleビジネスプロフィールの「口コミ」管理画面で返信の状況を確認できます。このキットが直接的な集客・評価向上を保証するものではありません。
