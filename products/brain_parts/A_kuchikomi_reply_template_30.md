## はじめに：口コミ返信、後回しにしていませんか？

「お礼を書かなきゃと思いながら、何日も放置してしまっている」

「ネガティブな口コミは、どう返せばいいかわからなくて見て見ぬふりをしている」

「返信はしているけど、毎回ゼロから考えるのが疲れる」

こういった悩みを持つ実店舗オーナーは、少なくありません。
Googleの口コミに返信することは義務ではありませんが、
**返信をしていない店舗より、丁寧に返信している店舗のほうが
次のお客様に好印象を与えやすい**という傾向があります。

ただ、毎回オリジナルの文章を書くのは時間がかかります。
このテンプレート集は、「書くこと」の負担を減らし、
返信をもっとラクに継続できるようにするために作りました。

## 口コミ返信を放置すると起きること

口コミへの返信がない状態は、次のお客様に「反応しない店舗」という印象を与えます。

とくに以下の状況では影響が大きくなりがちです。

- 競合他店が丁寧に返信しているとき
- 初めてその店を検討しているお客様が口コミを読んでいるとき
- ネガティブな口コミが返信なしのまま残っているとき

「口コミに返信することで集客が増える」「評価が上がる」とは断言できませんが、
返信を丁寧に続けることは、**既存客への感謝の表明**として機能します。

## やってはいけないNG返信 3パターン

テンプレートを使う前に、まずこれだけ確認してください。

### NG①：コピペしたまま固有名詞を入れ忘れる

```
×「[店舗名]へのご来店ありがとうございます」と書いたまま投稿
```
必ず `[ ]` の箇所を実際の情報に書き換えてから投稿してください。

### NG②：ネガティブ口コミに感情的に返信する

```
×「事実と異なります」「そのようなことはございません」と強く否定する
×「ご来店をお断りします」と返信する
```
感情的な返信は、他のお客様に悪い印象を与えます。
このテンプレート集のカテゴリCでは、誠実な対応例を用意しています。

### NG③：口コミ投稿を誘導・依頼する返信をする

```
×「ぜひまた口コミを書いてください」
×「次回ご来店時にクーポンをお渡しします（返信文内で口コミへの返礼を示唆）」
```
口コミの投稿を報酬で誘導する行為はGoogle利用規約に違反します。
このテンプレートにはそのような表現は含まれていませんが、ご自身でカスタマイズする際もご注意ください。

## よくある質問

**Q：Google以外の口コミにも使えますか？**
A：基本的な文体はYelp・食べログ・Googleマップ等に流用できます。
  プラットフォームごとに文字数制限が異なるためご調整ください。

**Q：複数店舗に使えますか？**
A：購入者ご本人の店舗であれば、複数店舗でご利用いただけます。

**Q：テンプレートは編集できますか？**
A：はい。`[ ]` 内の書き換えに加え、文体・内容もご自由に調整してください。

**Q：返信後に効果を確認する方法はありますか？**
A：Googleビジネスプロフィールの「口コミ」管理画面で返信の状況を確認できます。
  効果の計測はご自身のビジネス状況とあわせてご判断ください。
  このテンプレートが直接的な集客・評価向上を保証するものではありません。

**Q：ネガティブ口コミへの返信が怖いです。**
A：カテゴリCに誠実な返信テンプレートを用意しています。
  感情的にならず、事実を丁寧に伝えることを基本としています。
  深刻なトラブルに発展している場合は、専門家（弁護士等）への相談をお勧めします。


## よくある詰まりと対策

**「テンプレートが堅すぎて自分らしくない」**

→ 最初の1行だけ自分の言葉にして、あとはテンプレートを使うとバランスが取れます。
  例：「先日はありがとうございました！〇〇様のお言葉にスタッフ一同喜んでおります。…」

**「ネガティブ口コミにどのカテゴリを使えばいいかわからない」**

→ 内容で分岐します：
  - 待ち時間 → テンプレート21
  - スタッフ対応 → テンプレート22
  - 品質 → テンプレート23
  - 衛生 → テンプレート24
  - 予約ミス → テンプレート25
  - 事実が違うかも → テンプレート26
  - その他（詳細を聞きたい） → テンプレート27

**「同じお客様が複数回来てくれているのに毎回同じ返信になる」**

→ テンプレート07をベースに、「先日もお越しいただきありがとうございました」などの
  一言を加えると変化が出ます。

**「英語の口コミが来た。日本語で返していいか？」**

→ 可能であれば英語で返信するか、日本語と英語の両方を載せると親切です。
  このテンプレートは日本語専用です。英語対応は別途ご用意ください。
