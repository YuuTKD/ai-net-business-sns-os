---

## この商品でできること

Google口コミへの返信文を30本のテンプレートから選んで
コピー&ペーストするだけで、プロらしい返信が完成します。

- 高評価口コミへの感謝返信
- 普通〜良い口コミへの丁寧な返信
- 改善要望・ネガティブ口コミへの誠実な対応
- 業種別カスタムテンプレ（美容室・飲食・整体）

---

## 使い方

1. 口コミの内容・評価に合ったカテゴリを選ぶ
2. テンプレートをコピーする
3. `[ ]` で囲まれた箇所を実際の情報に書き換える
4. 口コミ管理画面（Googleビジネスプロフィール）に貼り付けて投稿

**書き換え箇所の例：**
- `[店舗名]` → あなたの店舗名
- `[スタッフ名]` → 担当スタッフの名前
- `[料理名/施術名]` → 口コミに書かれたメニュー名
- `[電話番号]` → 店舗の電話番号

## カスタマイズのコツ

### 書き換え必須箇所チェックリスト

```
[ ] [店舗名] → 実際の店舗名に変更
[ ] [ニックネーム] → 口コミ投稿者のお名前またはニックネームに変更
[ ] [スタッフ名] → 担当スタッフ名に変更（テンプレ9・28）
[ ] [料理名/施術名/メニュー名] → 口コミに書かれた内容に合わせて変更
[ ] [電話番号/メールアドレス] → 実際の連絡先に変更
[ ] [記念日の内容] → お誕生日・記念日など口コミの内容に合わせて変更
[ ] [改善取組内容] → テンプレ15・23などは具体的な改善内容を記入
[ ] [こだわりのポイント] → テンプレ18・29・30は自店のこだわりを記入
```

### 返信時の注意点

```
✓ 個人情報（フルネーム・住所・電話番号）は返信文に含めない
✓ 「必ず改善します」「絶対に」などの断定表現は避ける
  → 「改善に取り組んでまいります」と表現する
✓ ネガティブ口コミへの返信は感情的にならず、事実を淡々と記載する
✓ 謝罪が必要な場合は早めに投稿する（48時間以内を目安）
✓ 返信は250〜400文字程度が読まれやすい
```
