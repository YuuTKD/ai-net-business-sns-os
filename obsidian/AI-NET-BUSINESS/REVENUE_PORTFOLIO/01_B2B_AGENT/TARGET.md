# B2B_AGENT — TARGET ターゲット設計

---

## メインターゲット

**業種:** 美容サロン / ネイルサロン / 整体院 / 飲食店（カフェ含む） / 雑貨店 / 花屋

**状況:**
- SNSを「更新できていない」または「やってみたが続かない」
- Google口コミ返信が溜まっている
- ホットペッパーや食べログに依存していて自前集客がない
- AIに興味はあるが何をすれば良いかわからない
- 従業員数: 1〜5名（個人〜小規模）

---

## ターゲット分類

| 分類 | 優先 | 特徴 |
|---|---|---|
| 本命 | ★★★ | 実店舗あり / SNS集客悩み発信あり / AI活用に関心あり |
| サブ | ★★☆ | 実店舗あり / 更新停止中 / 悩み投稿なし |
| 保留 | ★☆☆ | オンラインのみ / 発信が少ない |
| 除外 | — | SNS運用代行業者 / MEO業者 / 競合 |

---

## ターゲットが使う言葉（検索・投稿キーワード）

```
「投稿ネタがない」「SNSが続かない」「口コミ返信が後回し」
「ホットペッパー やめたい」「Google投稿 やり方」「AI 集客 どうやって」
「予約が来ない」「Instagram更新できていない」「集客どうしたらいいか」
```
