# B2B_AGENT — COMPLIANCE コンプライアンス確認

---

## チェックリスト（投稿・資料作成時に毎回確認）

```
[ ] 成果保証表現がないか（「必ず」「確実に」「絶対に」）
[ ] 収益保証がないか（「月収〇万円」「誰でも稼げる」）
[ ] Google順位保証がないか（「検索1位になります」）
[ ] 完全無料 / 永久無料の表現がないか
[ ] 「AIが全部やってくれる」等の過大表現がないか
[ ] 診断結果レポートに医療・法的アドバイスが含まれていないか
[ ] アフィリエイト表記は必要か（Brain販売は自社商品として扱う）
[ ] 個人情報（顧客名・店名）をSNSに無断掲載していないか
```

---

## 使ってよい表現

```
✓ 「30分で今の集客状況を一緒に整理しましょう」
✓ 「実践した方から〇〇のような声をいただいています」（実績がある場合）
✓ 「一つひとつ確認しながら改善策をお伝えします」
✓ 「すぐ試せる方法を1〜3つお伝えします」
```

---

## 使ってはいけない表現

```
✗ 「必ず集客できます」
✗ 「AIで集客が自動化します」
✗ 「売上が確実にアップします」
✗ 「誰でも簡単に月商〇〇万円」
✗ 「完全無料サポート」
✗ 「Google検索で必ず上位になります」
```

## 個人情報取り扱い

- 診断申込みフォームで取得した情報はサービス提供のみに使用
- 第三者に提供しない
- 事例紹介時は必ず匿名化・許可取り
