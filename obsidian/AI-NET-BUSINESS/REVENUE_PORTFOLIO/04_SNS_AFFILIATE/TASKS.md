## 楽天/Amazon 補助導線（2026-07-10追加）

### 目的

Brain商品への流入を増やしつつ、実店舗オーナー向けグッズ紹介でアフィリエイト収入を追加。
Brain販売の邪魔にならない補助導線として活用。

### 掲載予定グッズ（審査通過後に掲載）

```
優先度順:
[ ] QRコードスタンド（Googleレビュー誘導・SNSフォロー）
[ ] ラミネーター・ポップ印刷用紙（店内POP自作）
[ ] タブレットスタンド（デジタルメニュー・アンケート用）
[ ] 店舗集客・SNS関連書籍
```

### やること（順番を守ること）

```
Step1 【人間が実施】楽天アフィリエイト or Amazonアソシエイト申請
Step2 【人間が実施】審査通過後、アフィリエイトURLを取得
Step3 【Claude Code】紹介文草案生成・コンプライアンス確認
Step4 【人間が確認】内容・表記を最終確認
Step5 【人間が実施】Brain商品内「おすすめグッズ」セクションに追加
Step6 【人間が実施】SNS投稿（手動・週1〜2回・#PR付き）
```

### アフィリエイト必須表記（これなしで掲載禁止）

```
記事冒頭: 「※この記事にはアフィリエイトリンクが含まれます」
Amazon時: 「Amazonのアソシエイトとして、[サイト名/屋号]は
           適格販売により収入を得ています」
           ※公開時は[サイト名/屋号]を実際の名称に変更（Amazon規約第5条）
楽天時:   「楽天アフィリエイトを通じた広告収入を得ています」
SNS投稿:  末尾に #PR を必ず追記（ステマ規制2023対応）
HTML:     rel="sponsored" または rel="nofollow"
```

### やってはいけないこと

```
× 審査前にリンクを掲載する（規約違反）
× #PR なしでアフィリエイト商品を紹介する（景品表示法違反）
× 「この商品を使えば必ず集客できます」等の保証表現
× rel=sponsored/nofollow なしのHTMLリンク（Googleガイドライン違反リスク）
× Claude Code がリンクを直接コードに埋め込む（人間が実施）
```
