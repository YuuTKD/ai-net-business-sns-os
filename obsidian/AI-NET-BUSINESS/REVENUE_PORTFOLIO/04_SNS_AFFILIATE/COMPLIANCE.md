## アフィリエイト表記ルール（2026-07-10更新）

| 状況 | 表記方法 |
|---|---|
| Brain商品（自社） | 表記不要（自社商品として明示） |
| 他社アフィリエイト全般 | 「#PR」または「※この投稿にはアフィリエイトリンクが含まれます」 |
| Amazonアソシエイト | 「Amazonアソシエイト・プログラムの参加者として、適格販売により収入を得ています」 |
| 楽天アフィリエイト | 「楽天アフィリエイトを通じた広告収入を得ています」 |
| ブログ・記事冒頭 | 「※この記事にはアフィリエイトリンクが含まれます」を必ず冒頭に |
| HTMLリンク | `rel="sponsored"` または `rel="nofollow"` を必ず付与 |
| SNS投稿 | 末尾に `#PR` を必ず記載（ステマ規制2023対応） |

### rel=sponsored / rel=nofollow について

```
Google の指針:
- アフィリエイトリンクや広告リンクには rel="sponsored" を付与する
- rel="nofollow" も代替として使用可能
- これを付けないとGoogleのガイドライン違反・ペナルティリスクがある

適用タイミング:
- Brain外にリンクを掲載するすべての外部アフィリエイトリンク
- 楽天・Amazon・ASP経由のすべてのリンク
```

### 注意: 審査前の掲載禁止

```
⚠️ 楽天/Amazonアソシエイト審査通過前にリンクを掲載することは規約違反
⚠️ 申請から審査通過まで待ってから掲載すること
⚠️ 掲載前に必ずゆうさんが最終確認
```
