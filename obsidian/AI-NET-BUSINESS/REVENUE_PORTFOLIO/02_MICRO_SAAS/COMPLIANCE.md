# MICRO_SAAS — COMPLIANCE

---

## チェックリスト

```
[ ] 「誰でも自動で投稿できます」等の過大表現がないか
[ ] 成果保証がないか
[ ] 「完全無料」「永久無料」を誤解を招く使い方をしていないか
[ ] API利用規約（Gemini / ChatGPT）に違反していないか
[ ] ユーザーデータを無断で記録・利用していないか
[ ] アフィリエイト表記が必要な場合に記載されているか
```

---

## 使ってよい表現

```
✓ 「業種とテーマを入力するだけで投稿文の案が出てきます」
✓ 「あくまで案なので、自分らしく調整してから使ってください」
✓ 「無料でお試しいただけます」
```

---

## 禁止表現

```
✗ 「AIが全部書いてくれます」（人間の確認不要と誤解させる表現）
✗ 「これを使えば必ずフォロワーが増えます」
✗ 「完全自動で集客できます」
```

---

## API利用規約確認

- Gemini API: 商用利用可（利用規約確認後に本番実行）
- ChatGPT: 生成コンテンツの利用規約確認
- Google Sheets / GAS: 利用規約確認済み（通常利用範囲内）
